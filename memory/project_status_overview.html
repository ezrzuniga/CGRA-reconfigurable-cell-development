How this maps to the image

Image block: Memory Cell
This is already represented by the implemented PE memory cell, SRAM bank, DMA engine, and NoC sockets.
Image block: Routing Cell / NoC
The simple interconnect is a first step toward the routing cell shown in the reference. It is functional but not yet a full router with credit-based flow control.
Image block: Scalar / Vector Cells
These exist in the repository as standalone PE modules, but they are not yet connected to the memory tile in a unified top-level simulation.
Image block: RISC-V Host / Bridge
The host-style CSR access is implemented through testbenches; a full CPU bridge is still pending.
